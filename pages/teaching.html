Teaching Computer Science

Beginning in September 2020, I taught two-classes at Champlain College St. Lambert. Champlain College is a CEGEP, or pre-university program for students in Quebec. The students are aged 17-19. The classes I taught were: Java Object Oriented Programming I and Computer IT fundamentals.

With the COVID-19 Pandemic, these courses were given 100% online. Please see the course resources here Java Object Oriented Programming I and IT fundamentals

The course focused on:

    • Variables
    • Conditionals
  • Classes
    • User-defined Classes
    • Scanner, Random, etc.
  • Looping
    • For, while, do while
    • User validation
  • Arrays
    • 1D Arrays
    • 2D Arrays
    • ArrayList
    • Sorting/Other operations on arrays
